# Restaurant Booking System

A MERN stack restaurant reservation system with a React frontend, Express API, MongoDB database, and booking management dashboard.

## Features

- Create restaurant table reservations
- View all bookings
- Search bookings
- Filter by status
- Update booking status
- Delete bookings
- MongoDB database support
- Responsive UI

## Tech Stack

- React
- Vite
- Node.js
- Express.js
- MongoDB
- Mongoose

## Run Locally

```bash
npm run install:all
npm run dev
